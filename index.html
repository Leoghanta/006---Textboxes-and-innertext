Workshop 6: Using Textboxes for Input and InnerText for Output

Workshop 6

Welcome to Workshop 6! Today, we'll be diving into an essential aspect of JavaScript and DOM manipulation: using textboxes for input and innerText for output. By the end of this lesson, you'll understand how to take user input from a textbox, process it using JavaScript, and display the result on your web page using innerText.

Introduction to Textboxes in HTML

Textboxes are one of the most common ways to receive input from users. In HTML, a textbox is created using the <input> tag with the type attribute set to "text".

Example:


<input type="text" id="userInput" placeholder="Enter your text here">

id="userInput": The id attribute is crucial because it allows JavaScript to easily reference and manipulate this element.

placeholder="Enter your text here": The placeholder text shows up inside the textbox as a hint for the user.

Introduction to Buttons in HTML

To accept user input, buttons are good triggers. We can create onclick listeners and call a piece of code. Buttons are created using the <button> tag.

Example:


<button id="ouchButton" onclick="alert('Ouch!');">Click Me!</button>

id="ouchButton": The id attribute will allow JavaScript to easily reference and manipulate this element, and set up listeners.

onclick="alert('Ouch!');": The onclick will run a piece of code when a click is detected.

The innerText Property in JavaScript

Once we have user input, we typically want to display some sort of feedback or result on the page. This is where the innerText property of HTML elements comes into play.

What is innerText?

'innerText' allows you to get or set the text content inside an HTML element.

Example:


<p id="outputText">Your result will be displayed here.</p>

To change the content of this

element using JavaScript:


document.getElementById("outputText").innerText = "Hello, World!";

The id of this paragraph is outputText. Use JavaScript to change the text.

Capturing User Input with JavaScript

Now, let's combine what we've learned. We'll capture the user's input from the textbox and display it using innerText.

Step-by-Step Example:

1.HTML Structure:


<input type="text" id="userInput" placeholder="Enter your text here">
<button onclick="displayInput()">Submit</button>
<p id="resultText">Your result will be displayed here.</p>

2. JavaScript Function:


function displayInput() {
    // Get the value from the textbox
    let userInput = document.getElementById("userInput").value;

    // Set the value to the paragraph
    document.getElementById("resultText").innerText = userInput;
}

In this example:

✔️ The user types into the textbox.

✔️ When the "Submit" button is clicked, the displayInput function is triggered.

✔️ The function captures the text from the textbox and assigns it to the innerText of the <p> element.

A Deeper Look: Common Use Cases

Let's explore a few variations of how you can use textboxes and innerText together.

Greeting Message


<input type="text" id="userName" placeholder="Enter your name">
<button onclick="greetUser()">Greet Me</button>
<p id="greetingMessage"></p>

<script>
    function greetUser() {
        let name = document.getElementById("userName").value;
        document.getElementById("greetingMessage").innerText = "Hello, " + name + "!";
    }
</script>

This script greets the user by name after they input it into the textbox.

Example 2: Calculating a Simple Sum


<input type="text" id="number1" placeholder="Enter first number">
<input type="text" id="number2" placeholder="Enter second number">
<button onclick="calculateSum()">Add Numbers</button>
<p id="sumResult"></p>

<script>
    function calculateSum() {
        let num1 = parseInt(document.getElementById("number1").value);
        let num2 = parseInt(document.getElementById("number2").value);
        let sum = num1 + num2;
        document.getElementById("sumResult").innerText = "The sum is: " + sum;
    }
</script>

In this example, the user enters two numbers, and the script calculates and displays their sum.

Example 3: Simple Text Transformation


<input type="text" id="userText" placeholder="Enter your text">
<button onclick="transformText()">Uppercase</button>
<p id="transformedText"></p>

<script>
    function transformText() {
        let text = document.getElementById("userText").value;
        let transformed = text.toUpperCase();
        document.getElementById("transformedText").innerText = transformed;
    }
</script>

Here, the script converts the user's text input to uppercase and displays it.

Q&A and Troubleshooting

  • ✅ Text not changing? Make sure you spell your id correctly, and that the case is the same.
  • ✅ NaN appears when trying to do maths with the wrong data type. use parseInt() to convert strings to integers.
  • ✅ Debugging: You can add a conditional check to ensure the input is not empty before updating the innerText.
